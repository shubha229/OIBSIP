# OIBSIP — BMI Calculator Project

Contains:
- `bmi/`
  - `bmi_cli.py` — Command-line BMI calculator
  - `bmi_gui.py` — Tkinter GUI calculator that saves history to `bmi_history.csv`
  - `bmi_plot.py` — Plot BMI history from CSV using pandas + matplotlib
